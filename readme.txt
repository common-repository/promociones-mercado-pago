Lists Mercado Pago credit and debit card active promotions.

== Description ==

Lists Mercado Pago credit and debit card active promotions using a responsive flex grid.

This is not an official Mercado Pago plugin.

== Installation ==

1. Install like any other plugin, directly from your plugins page.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the shortcode [promociones-mercado-pago] in the post or page you want to see the promotions.

== Frequently Asked Questions ==

= Is this plugin made or maintain by Mercado Pago official team? =

No. It's just a small contribution of a Mercado Pago user.

= Is it possible to integrate the shortcode with WPBakery Page Builder? =

Yes just navigate to Shortcode Mapper link inside WPBakery Page Builder and add the shortcode there.
